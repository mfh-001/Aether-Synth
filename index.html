Procedural Ambient Synthesizer

Natural language mood descriptions mapped to DSP synthesis parameters in real-time. Multi-layer additive synthesis, convolution reverb, and LFO modulation — entirely in the browser via Web Audio API.

⚙ Signal Chain Architecture
Layer 1 — NLP Tokenizer

Mood text is tokenized against a weighted keyword lexicon. Each token scores against 12 acoustic dimensions: brightness, roughness, depth, warmth, tension, density, spaciousness, rhythm, and harmonic complexity. Scores are normalized and fed as continuous parameters to the synthesis engine.

Layer 2 — DSP Parameter Map

NLP scores deterministically configure: root frequency (40–220 Hz), modal scale intervals, reverb decay (1–12s), pad filter cutoff, LFO rate & depth, noise color, sub frequency, and per-layer amplitude. 12 parameters, all continuous.

Layer 3 — Web Audio Graph

5-layer additive synthesis graph: Sub (sine, fundamental), Pad (sawtooth+triangle stack, detuned ±6 cents), Shimmer (FM-modulated sine), Texture (bandpass-filtered noise), Pulse (LFO-tremolo triangle). All routed through convolution IR reverb.

Technical Implementation
NLP → DSP Parameter Engine

The mood interpreter uses a weighted token-matching approach. Input text is lowercased and split into tokens, each scored against a hand-crafted acoustic lexicon of ~80 keyword clusters (dark/ominous, bright/crystalline, aquatic, spatial, rhythmic, etc.).

Scores are interpolated across 12 continuous synthesis dimensions using weighted sums, producing parameters like rootFreq, reverbTime, noiseColor, and pulseRate in valid DSP ranges.

Web Audio API Synthesis Graph

Uses the browser's AudioContext with 5 oscillator chains running in parallel. The pad layer stacks 25 oscillators (5 scale tones × 5 detune offsets of ±6 cents) through a shared lowpass filter, creating the characteristic lush pad texture.

LFO modulation is applied to the shimmer oscillator's frequency input, creating natural pitch vibrato. All layers merge at a master gain node before convolution reverb.

Convolution Reverb (IR Synthesis)

Rather than loading an impulse response file, the reverb IR is procedurally generated: exponentially decaying white noise synthesized directly into an AudioBuffer. Decay time τ maps from 1–12 seconds based on the "spaciousness" score.

This eliminates external assets entirely — the IR for a 12-second cathedral reverb is generated in ~5ms and occupies ~1MB of RAM.

Offline Rendering & WAV Export

OfflineAudioContext renders audio faster than real-time (a 5-minute stereo file at 44.1kHz takes ~8 seconds to render on a mid-range CPU). The rendered AudioBuffer is then encoded to 16-bit PCM WAV format via a hand-written encoder — no external libraries needed.

Output: 44.1kHz, 16-bit, stereo WAV. Compatible with all DAWs and video editors.

Scale Theory — Modal Harmony

The scale parameter is an array of semitone intervals from the root note. Different emotional qualities map to different modal scales:

// NLP mood → modal scale mapping dark/ominous → [0, 3, 5, 8, 10] // Dorian mode (minor with raised 6th) tense/thriller → [0, 1, 5, 6, 10] // Phrygian dominant fragments bright/open → [0, 2, 4, 7, 9] // Pentatonic major mysterious → [0, 1, 5, 7, 10] // Locrian fragments neutral/drift → [0, 2, 5, 7, 10] // Pentatonic minor (default) // Frequency calculation (equal temperament) freq(root, interval, octave) = root × 2^((octave×12 + interval) / 12)
